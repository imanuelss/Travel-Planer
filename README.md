# Aplikasi Travel Planer
Aplikasi Travel Planer Dibangun menggunakan Golang dengan thema Wonderlog.com Clone API .

# Project Concept
Tujuan Aplikasi membantu user untuk :
- Register / login
- Cari destinasi
- Bikin trip
- Tambah itinerary per hari
- Generate AI itinerary
- Simpan budget trip
